# Routing Algorithms

This folder contains Java implementations and simulations of major routing algorithms and protocols.

### Key Topics Covered:
- **Adaptive Routing Algorithms:** Isolated, Centralized (RCC), and Distributed (Distance Vector Routing)
- **Non-Adaptive Routing Algorithms:** Flooding and Random Walk
- **Border Gateway Protocol (BGP):** Path Vector Routing simulation
- **Open Shortest Path First (OSPF):** Link State Routing using Dijkstra's Algorithm
