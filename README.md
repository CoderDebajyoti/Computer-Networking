## How to Run

1. Navigate to the ASSIGNMENT1 directory:
   ```
   cd ASSIGNMENT1
   ```

2. Compile a Java file (replace `A1_P1.java` with the desired file):
   ```
   javac A1_P1.java
   ```

3. Run the compiled program:
   ```
   java A1_P1
   ```

Note: Some programs require user input via the console.

## RSA Encryption

The TEST folder contains a Java program (RSA.java) that demonstrates RSA encryption and decryption:

- **RSA.java**: Implements RSA encryption and decryption using helper methods for calculating the greatest common divisor and computing the modular inverse.

## How to Run RSA Program

1. Navigate to the TEST directory:
   ```
   cd TEST
   ```

2. Compile the RSA program:
   ```
   javac RSA.java
   ```

3. Run the compiled program:
